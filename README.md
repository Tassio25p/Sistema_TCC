# 🎨 Artifolio

Sistema web para compra, venda e gerenciamento de obras digitais, conectando artistas e clientes em uma plataforma moderna e escalável.

---

## 📌 Sobre o Projeto

O **Artifolio** é uma plataforma desenvolvida como Trabalho de Conclusão de Curso (TCC), com o objetivo de permitir que artistas publiquem suas obras digitais e clientes possam adquiri-las de forma simples e segura.

O sistema contempla desde a publicação da arte até o pagamento, cálculo de comissão e entrega do arquivo ao cliente.

---

## 🚀 Funcionalidades

### 🔐 Autenticação

* Cadastro de usuários
* Login e autenticação
* Diferenciação entre cliente e artista

### 🎨 Artistas

* Criação de perfil comercial
* Gerenciamento de obras
* Visualização de ganhos

### 🖼️ Obras (Postagens)

* Cadastro de obras com imagem
* Definição de título, descrição e preço
* Listagem e visualização

### 🛒 Pedidos

* Compra de obras
* Acompanhamento de status

### 💳 Pagamentos

* Integração com API de pagamento
* Registro de transações

### 💰 Comissão

* Cálculo automático da taxa da plataforma
* Controle de valores brutos e líquidos

### 🚚 Entrega Digital

* Disponibilização do arquivo ao cliente
* Registro de entrega

---
### 🔧 Backend

* Arquitetura em camadas:

  * Controller
  * Service
  * Repository
* API REST

### 💻 Frontend

* Interface web moderna
* Consumo de API

## 🧱 Tecnologias Utilizadas

### Backend

* API REST
* PostgreSQL
* Arquitetura em camadas

### Frontend

* React
* Tailwind CSS

### Infraestrutura

* CDN para imagens
* Cache (Redis)
* Integração com API de pagamento
